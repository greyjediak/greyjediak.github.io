Electronic Nose

Sensors | Embedded | PCB Design | ML (pending)

Objectives

  • Develop a low-cost electronic nose by interfacing six sensors to a microcontroller
  • Read sensor data and interpret it into meaningful units
  • Provide some insight into underlying health conditions that breath samples indicate

Current tasks: Designing and building a breath acetone sensing system. Involves PCB design, programming in the Arduino IDE. Potentially, ML will be implemented to differentiate between samples when integrating all sensors together.

Acetone Sensor Schematic Acetone PCB Layout

Goal: Interface the TGS sensor with the Seeed Xiao nRF52480 Plus. Since response time is ~40s, and our intended target response time is 20s, early inference will be applied to the transient response. The TGS is a MEMS hot-wire sensor and the bridge voltage difference is used to calibrate acetone readings.
